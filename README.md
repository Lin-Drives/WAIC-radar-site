# 智讯雷达 · WAIC 2026 实时动态聚合站

> **不生产新闻，只让每一条 WAIC 动态都有据可查。**  
> Generated by KIMI K3 agent swarm, and deployed to Cloudflare Pages for demonstration.

---

## 🌊 这是什么？

**智讯雷达** 是一个为 [WAIC 2026 世界人工智能大会](https://www.worldaic.com.cn/)（World Artificial Intelligence Conference）打造的实时新闻聚合导航站。

在这个信息过载的时代，AI 大会每天都有几十条新动态散落在各处——官网公告、财经媒体、科技博客、行业公众号……**你需要一个雷达，把这些信号全都收到一屏里。**

这就是「智讯雷达」在做的事。

## 🛠️ 本地运行

```bash
# 克隆仓库
git clone https://github.com/Lin-Drives/WAIC-radar-site.git
cd WAIC-radar-site

# 进入前端源码目录（如果你有源码的话）
# npm install
# npm run dev

# 或者直接打开部署产物
cd deployment
# 用任意静态服务器打开，例如：
npx serve .
# 或 Python 内置服务器：
# python3 -m http.server 8080
```

## 📝 收录规则

每一条新闻进入雷达，都要过四关：

```
① 采集 → 每日巡检 30+ 权威信源
② 核实 → 逐条人工打开验证链接
③ 归类 → 人工标注七大分类
④ 发布 → 去重、按时间排序上架
```

**我们承诺：** 热度值按信源权威度 × 时效综合打分，URL 去重，一律采用原文发布时间。不会为了流量编造标题。
